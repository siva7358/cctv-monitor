# CCTV Camera Status - Real-Time Monitoring System

A real-time CCTV camera monitoring dashboard with IP ping status tracking.

## Features

- 🗺️ Interactive map with 180+ camera locations
- 🔴 Real-time IP pinging (every 5 seconds)
- 🟢 Online/Offline/Partial status indicators
- 🔍 Search by JB number, IP, or location
- 📊 Live statistics dashboard
- 📱 Fully responsive design

## Technologies Used

- HTML5 / CSS3 / JavaScript
- Leaflet.js for maps
- CartoDB tiles for map rendering

## How It Works

1. All IPs are pinged simultaneously every 5 seconds
2. Status updates in real-time on the map and device list
3. Click any marker to see detailed IP status
4. Filter by Online, Offline, or Partial status

## Setup

Simply open `index.html` in any modern browser, or deploy to GitHub Pages.
